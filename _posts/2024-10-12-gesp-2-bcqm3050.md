## BCQM3050

### 题目要求

#### 描述

>输入一个浮点数，将其向零舍入到整数。说明：向零舍入的含义是，正数向下舍入，负数向上舍入。

#### 输入

>一个浮点数。

#### 输出

>一个整数，即向零舍入到整数的结果。

#### 输入样例

>2.3

#### 输出样例

>2

---

### 题目分析

根据题目要求，C++中`double`型转换成`int`型的处理方式就是题目要求，因此我把他归类为二级数据类型转换的题目。

#### 代码参考

```cpp
#include <iostream>
using namespace std;
int main() {
    double a;
    cin >> a;
    cout << (int)a;
    return 0;
}
```
