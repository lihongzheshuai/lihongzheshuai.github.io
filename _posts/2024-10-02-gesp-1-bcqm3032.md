## BCQM3032

### 题目要求

#### 描述

>在梯形中阴影部分面积是150平方厘米，求梯形面积。
![GESP3032](/images/post/gesp/3032.png)

#### 输入

>无。

#### 输出

>输出梯形面积（保留两位小数）。

#### 输入样例

>无

#### 输出样例

>无

---

### 题目分析

看图计算。用已知阴影三角形的面积求出三角形也是梯形的高，然后根据梯形面积公式求出梯形面积即可。

#### 代码参考

```cpp
#include <cstdio>
int main() {
    double h = 0;
    h = 150.0 * 2 / 15;
    printf("%.2f", (15 + 25) * h / 2);
    return 0;
}
```
