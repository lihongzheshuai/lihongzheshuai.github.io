## BCQM3031

### 题目要求

#### 描述

>已知小明有n元，他买了一本书，这本书原价为m元，现在打8折出售。求小明还剩多少钱(保留2位小数)。

#### 输入

>输入n,m。

#### 输出

>小明还剩多少钱(保留2位小数)。

#### 输入样例

>100 100

#### 输出样例

```console
20.00
```

---

### 题目分析

没想到这道题对我孩子的难点居然是不知道什么是8折，小朋友确实没自己买过什么东西。8折就是：*价格 * 0.8*。知道这个关系剩下就是简单的计算和按格式输出了。浮点数格式化***%m.nf***知识点，已在[BCQM3027](https://www.coderli.com/gesp-1-bcqm3027/)中详细介绍，不再赘述。

#### 代码参考

```cpp
#include <cstdio>
int main() {
    int n, m;
    scanf("%d%d", &n, &m);
    printf("%.2lf", n - m * 0.8);
    return 0;
}
```
