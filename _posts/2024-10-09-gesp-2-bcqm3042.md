## BCQM3042

### 题目要求

#### 描述

>我们知道每个字符都有一个对应的 ASCII 码。现在输入一个 ASCII 码，要求你输出对应的字符。

#### 输入

>一个整数，即字符的 ASCII 码，保证对应的字符为可见字符。

#### 输出

>一行，包含相应的字符。

#### 输入样例

>65

#### 输出样例

>A

---

### 题目分析

根据ASCII规则，只需要把输入的`int`类型变量，强转成`char`类型，直接输出即可。

#### 代码参考

```cpp
#include <iostream>
using namespace std;
int main() {
    int a;
    cin >> a;
    cout << (char)a;
    return 0;
}
```
