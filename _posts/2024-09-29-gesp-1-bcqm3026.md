## BCQM3026

### 题目要求

#### 描述

>求3个整数的和。输入a、b、c这3个整数，求它们的和。

#### 输入

>3个整数。

#### 输出

>三个数的和。

#### 输入样例

>1 2 3

#### 输出样例

```console
6
```

---

### 题目分析

如果做过之前的问题，难度就完全没有了。读入变量，计算，输出即可。连计算都变成最简单的加法了。

#### 代码参考

```cpp
#include <iostream>
using namespace std;
int main() {
    int a, b, c;
    cin >> a >> b >> c;
    cout << a + b + c;
    return 0;
}
```
