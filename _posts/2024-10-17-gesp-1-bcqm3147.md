## BCQM3147

### 题目要求

#### 描述

>利用for循环。计算输出1+2+3+...+n的和。

#### 输入

>输入n。对于100%的数据，1≤n≤100。

#### 输出

>如题述，之和。

#### 输入样例

>10

#### 输出样例

>55

---

### 题目分析

- 读入一个整型变量
- 定义一个变量，用于记录当前的和。
- 利用`for`循环，每遍历一个变量，就把当前变量+到当前的和上，直到遍历到最后一个变量，就是最终的和。

#### 代码参考

```cpp
#include <iostream>
using namespace std;
int main() {
    int n;
    cin >> n;
    int h = 0;
    for (int i = 1; i <= n; i++) {
        h += i;
    }
    cout << h;
    return 0;
}
```
