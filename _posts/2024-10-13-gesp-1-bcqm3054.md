## BCQM3054

### 题目要求

#### 描述

>利用公式 C=5×(F−32)÷9(其中C表示摄氏温度，F表示华氏温度)进行计算转化,输入华氏温度F，输出摄氏温度C，要求精确到小数点后5位。

#### 输入

>输入一行，包含一个实数F，表示华氏温度。（F≥−459.67）。

#### 输出

>输出一行，包含一个实数，表示对应的摄氏温度，要求精确到小数点后5位。

#### 输入样例

>41

#### 输出样例

>5.00000

---

### 题目分析

- 输入为实数，因此需要用`double`作为输入。
- 按照题目给出的计算公式代入计算
- 根据题目要求用printf进行格式化输出。保留5位小数，`%.5f`。

#### 代码参考

```cpp
#include <cstdio>
int main(){
    double f;
    scanf("%lf", &f);
    double c;
    c = 5 * (f - 32) / 9;
    printf("%.5f", c);
    return 0; 
}
```
